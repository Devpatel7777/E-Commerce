# E-Commerce Order Management System

## Objective
Developed using MySQL to manage products, customers, orders, payments, and shipping.

## Features
- CRUD Operations
- Joins & Subqueries
- Window Functions
- CASE Expressions
- Sales Analysis

## Database Tables
- Products
- Categories
- Customers
- Orders
- Order_Items
- Payments
- Shipping

## How to Run
1. Open MySQL
2. Run `ecommerce_management.sql`
